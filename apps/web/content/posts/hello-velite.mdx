# Velite 소개

Velite는 Next.js와 완벽하게 통합되는 type-safe content management 라이브러리입니다.
Markdown과 MDX 파일을 Zod 스키마 기반으로 검증하고, TypeScript 타입을 자동 생성해줍니다.

## 주요 특징

### 1. Type Safety

Zod 기반의 스키마 검증으로 컴파일 타임에 오류를 잡을 수 있습니다.

```typescript
const posts = defineCollection({
  name: "Post",
  pattern: "posts/**/*.mdx",
  schema: s.object({
    title: s.string().max(99),
    date: s.isodate(),
    tags: s.array(s.string()),
  }),
})
```

### 2. 자동 타입 생성

빌드 시 `.velite` 디렉토리에 타입 정의 파일이 자동으로 생성됩니다.

### 3. MDX 지원

Markdown뿐만 아니라 MDX도 완벽하게 지원하여, React 컴포넌트를 문서에 포함할 수 있습니다.

## 시작하기

Velite를 사용하면 복잡한 설정 없이도 강력한 블로그 시스템을 구축할 수 있습니다.
