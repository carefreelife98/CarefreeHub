# Cost / Billing

## AWS Cost Explorer - 비용 분석

청구서에서 Amazon EC2 비용 증가를 관찰하고 인스턴스 유형의 원치 않는 수직적 확장의 근본 원인을 식별하려면:

- **Cost Explorer**의 세분화된 필터링 기능을 사용하여 인스턴스 유형을 기반으로 EC2 비용에 대한 심층 분석을 수행

> **AWS Cost Explorer는 비용과 사용량을 보고 분석할 수 있는 도구**입니다.
> 기본 그래프, Cost Explorer 비용 및 사용량 보고서 또는 Cost Explorer RI 보고서를 사용하여 사용량 및 비용을 탐색할 수 있습니다.
