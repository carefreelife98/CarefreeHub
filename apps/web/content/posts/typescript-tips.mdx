# TypeScript 실무 활용 팁

TypeScript를 더 효과적으로 사용하는 방법들을 소개합니다.

## 1. Utility Types 활용

TypeScript의 내장 Utility Types를 적극 활용하세요.

```typescript
type User = {
  id: string
  name: string
  email: string
}

// 일부 필드만 optional로
type UserUpdate = Partial<User>

// 특정 필드만 선택
type UserPreview = Pick<User, "name" | "email">

// 특정 필드 제외
type UserWithoutId = Omit<User, "id">
```

## 2. Discriminated Unions

타입 안전성을 높이는 강력한 패턴입니다.

```typescript
type Success = {
  status: "success"
  data: string
}

type Error = {
  status: "error"
  message: string
}

type Result = Success | Error

function handleResult(result: Result) {
  if (result.status === "success") {
    console.log(result.data) // 타입 안전
  } else {
    console.error(result.message) // 타입 안전
  }
}
```

## 3. const assertions

불변성을 강제하고 더 정확한 타입 추론을 얻을 수 있습니다.

```typescript
const routes = ["home", "about", "contact"] as const
type Route = typeof routes[number] // "home" | "about" | "contact"
```

## 4. Template Literal Types

문자열 패턴을 타입으로 표현할 수 있습니다.

```typescript
type HttpMethod = "GET" | "POST" | "PUT" | "DELETE"
type Endpoint = `/api/${string}`
type ApiRoute = `${HttpMethod} ${Endpoint}`
```

## 마무리

TypeScript의 고급 기능들을 활용하면 더 안전하고 유지보수하기 쉬운 코드를 작성할 수 있습니다.
